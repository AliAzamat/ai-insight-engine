# AI Insight Engine with Python & OpenAI

A project that adds the AI layer to a data platform. A Python service that summarizes warehouse metrics into executive narratives, answers plain-English questions by generating SQL (text-to-SQL) against trusted marts, forecasts trends, retries and caches calls, and serves it all over FastAPI — with the structured outputs, guardrails, and observability that make LLMs safe and affordable over enterprise data.

## Stack
- Python
- OpenAI API
- LLM
- Pydantic
- Text-to-SQL
- Prompt Engineering
